# Exercícios de para

## Exercício 01
Solicitar 5 nomes das músicas favorita do usuário

Apresentar no final:
- Nome da música que contém a maior quantidade de caracteres
- Nome da música que contém a menor quantidade de caracteres

## Exercício 02
Solicitar os dados dos produtos:
- Nome
- Quantidade
- Preço unitário
- Largura
- Altura
- Comprimento

- Apresentar no final:
- O custo efetivo desse produto (qtd * preço)
- O nome do produto que contém a maior altura
- A média dos preços unitários
- A soma total dos custos efetivos do produto
